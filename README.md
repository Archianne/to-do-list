## To-do list

Simple to-do list, made with HTML, CSS and Vanilla JavaScript.

### Learn:

- `changing tabs`
- `querySelector()`
- `localStorage`
- `json.parse()`
- `json.stringify()`

### How to use

- Add items to the list
- Delete each item or the whole list
- Edit items
- Refresh the page and all items are restored
